# VS Code

### Open project on VS Code
```bash
cd 'project path'
code .
```

### Shortcuts

#### Copy line
<kbd>Shift</kbd> + <kbd>Option</kbd> + <kbd>Down</kbd> or <kbd>Up</kbd>
